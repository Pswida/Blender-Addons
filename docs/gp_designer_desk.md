# GP Rig Tools

Zaawansowane narzędzia do rigowania i kontroli Grease Pencil w Blenderze.

---

## Instalacja

1. Pobierz plik ZIP dodatku
2. Otwórz Blender
3. Przejdź do:

```text
Edit → Preferences → Add-ons → Install
```

4. Wybierz ZIP
5. Włącz addon

---

## Wymagania

| Blender | Status |
|---|---|
| 4.2+ | ✅ |
| 4.1 | ⚠ częściowo |
| 3.x | ❌ |
